# کامنت ها در bash 

اگر زبان برنامه نویسی دیگه ایی رو کار کرده باشید، میدونید که میتونید توی کداتون از کامنت استفاده کنید. تو پارت های قبلی هم یه اشاره ریزی بهش کردم! کامنت ها فقط برای برنامه نویس هستن و خود زبان اون قسمت هارو اجرا نمیکنه. در واقع اگه خواستید توضیحی یا راهنمایی برای خودتون یا بقیقه برنامه نویس ها بزارید از کامنت استفاده میکنیم.

برای اینکه توی bash از کامنت استفاده کنیم باید از کارکتر هشتگ که اینه `#` استفاده کنیم. توجه کنید که کامنت ها توی اجرا نمایش داده نمیشن!

این زیر یه مثال ساده ازش داریم:

```bash
# This is a comment and will not be rendered on the screen
# این یه کامنته که روی صفحه نمایش داده نمیشه
```

حالا میتونید اون فایل قبلیتون رو ویرایش کنید و برای خودتون یه کامنت بزارید:

```bash
#!/bin/bash

# Ask the user for their name

read -p "What is your name? " name

# Greet the user
echo "Hi there $name"
echo "Welcome to programming world!"
```

شاید فعلا اینجا استفاده درستی از کامنت ها نکرده باشیم! کامنت ها برای توضیح یه متغیر یا تابع به کار میرن که بدونیم کار اون تابع یا متغیر چیه. همچنین اگه کس دیگه ایی روی پروژمون کار کرد بفهمه چی کار کردیم.
