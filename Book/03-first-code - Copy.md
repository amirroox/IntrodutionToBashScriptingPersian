# اولین کد با Bash

حالا که فایلمونو ساختیم و اولش bash shebang رو هم گذاشتیم بریم سراغ اولین اسکریپتمون که بدرد جایی هم نمیخوره!

فایل fisrt.sh یا اون فایلی که ساختید رو باز کنید و یه خط فاصله بدید و مورد زیر رو بنویسید:

```bash
#!/bin/bash

echo "Hi Baby!"
```

دستور بالا یه چیزی توی ترمینال چاپ میکنه. فایلتون رو سیو کنید و بریم برای اجرای اسکریپت.

بعد از اینکه اسکریپت رو نوشتیم، باید به اون فایل اجازه اجرا شدن بدیم، چون به طور پیشفرض اجازه اجرا نداریم :(

```bash
chmod +x fisrt.sh
```

دستور بالا اجزای اجرا رو بهمون میده و x مخفف Execution عه. حالا به شکل زیر اسکریپت سادمون رو اجرا میکنیم:

```bash
./first.sh
```

خب همونطور که احتمال میدادید چیزی که نوشتیم روی ترمینال چاپ میشه.

راه دیگه ایی برای اجرای اسکریپت استفاده مستقیم از bash عه:

```bash
bash fisrt.sh
```

همچنین bash یه زبان تعاملیه، پس بدون اینکه فایل خاصی ایجاد کنید، همینجوری روی ترمینال دستور زیر رو بنویسید هم اجرا میشه!

```bash
echo "Hi Baby!"
```

زمانی که چندین دستور وابسته به هم هستن و کنار هم دیگه مفیدن اسکریپت نویسی کمک زیادی بهمون میکنه قشنگا! 

Putting a script together is useful once you have to combine multiple commands together.
