# متغیر ها در Bash

مثل هر زبون دیگه ایی توی بَش هم متغیر داریم، فقط باید توجه کنید که نوع داده نداریم و همه ی عدد ها و کارکتر ها کنار هم دارن زندگی میکنن. 

برای اینکه یه متغیر بسازید فقط نیازه اول اسم متغیرتون رو بنویسید و بعد `=` و بعد مقدارش رو وارد کنید

```bash
name="AmirRoox"
```

>این نکته رو در نظر داشته باشید که نمیتونید قبل و بعد از `=` هیچ فاصله ایی داشته باشید... پس الکی برای خوشگلی Space نزنید.

حالا که متغیر خوشگلمون رو ساختیم، برای اینکه بتونیم به مقدارش دسترسی داشته باشیم باید اول یه `$` بزاریم. اگه PHP کار کرده باشید این علامت دلار براتون آشناست.

```bash
echo $name
```

به طور کلی برای متغیر ها نیاز به براکت {} نیست، اما خیلی خوبه که ازشون استفاده کنید. هم متغیرهاتون رو بهتر شناسایی میکنید و هم بعضی جاها دچار مشکل نمیشید:

```bash
echo ${name}
```

کد بالا هم که خروجی `AmirRoox` رو داره چون از اول این مقدار رو بهش دادیم!

دستورات بالا رو توی ترمینال زدیم! حالا بریم رو اون فایل قبلی که داشتیم و این متغیرو تو اونم بزاریم

میتونید با nano یا vim یا هر ادیتور دیگه ایی فایلتون رو باز کنید

```bash
nano fisrt.sh
```

خودتون میتونید هرچیزی دوست داشتید جای متغیر و متن بزارید:

```bash
#!/bin/bash

name="AmirRoox"

echo "Hi World, My name is $name"
```

و دوباره از اول برید سیوش کنید و بعد اجراش کنید

```bash
./fisrt.sh
```

همچنین میتونید از هر تعداد متغیری که میخواید استفاده کنید و متغیر های قبلی رو هم میتونید خیلی راحت بازنویسی کنید:

```bash
#!/bin/bash

name="AmirRoox"
greeting="Hello"

echo "$greeting $name"

name="Dani"
echo "$greeting $name"
```

مثل زبان جاوا اسکریپت توی بَش هم میتونید آخر کد هاتون از سیمی کالن `;` استفاده کنید. گذاشتن یا نزاشتنش اجباری نیست اما بزارید بد نیست!


حالا بیاید یه کار جالب تر بکنیم! فرض کنید میخوایم از کاربر مقدار متغیر هارو بگیریم! اونوقت میتونید از روش زیر این کارو کنیم:

```bash
./fisrt.sh Amir Dani!
```
وقتی جلوی فایلمون مقدار های متفاوت بدیم یعنی به فایلمون پارامتر اضافه کردیم. الان توی دستور بالا پارامتر اول Amir و پارامتر دوم Dani! به فایلمون ارسال میشه. توی فایلمون باید بتونیم این پارامتر هارو بخونیم:

```bash
#!/bin/bash

echo "Hello there" $1

```

توی دستور بالا `$1` رو میبنید. این اولین ورودیه ماست! شما هرچقدر که بخواید میتونید جلوی فایلتون پارامتر پاس بدید و همه اونا به ترتیب با $1 و $2 و $3 و الی آخر در دسترس هستن. همچنین میتونید از `$@` استفاده کنید تا همه ورودی هارو بخونید!

حالا برای اینکه بهتر متوجه بشیم میتونیم فایلمون رو هم تغییر بدیم:

```bash
#!/bin/bash

echo "Hello there" $1

# $1 : پارامتر اولمون

echo "Hello there" $2

# $2 : پارامتر دوممون

echo "Hello there" $@

# $@ : همه پارامترا
```
حالا فایلمون رو با دستور زیر اجرا میکنیم:

```bash
./first.sh amir dani! ooh
``` 
خروجی میشه این:

```bash
Hello there amir
Hello there dani!
Hello there amir dani! ooh
```

اون قسمت هایی که با # توی کد وجود داره کامنت هستن و Bash اونارو اجرا نمیکنه!
